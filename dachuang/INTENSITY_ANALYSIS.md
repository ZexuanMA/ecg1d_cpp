# Fluorescent Spot Intensity Analysis (v2 — Grid-Based)

## Overview

从三张 BMP 荧光图像（0μg/L, 5μg/L, 20μg/L）中提取离散化的光强分布数据，用于 Ridge-Projection 算法求解转移矩阵 $T$。

图像为红色荧光传感器图像（3536×3536 RGB），有效信号在 **红色通道**（R channel），G/B 通道近零。

**v2 改进**：利用传感器底板的**规则网格结构**进行发光点定位，替代 v1 的通用局部极大值检测。增加分区质量过滤，排除噪声/伪影区域。

## Step 1: 网格结构检测

**观察**：荧光发光点排列在规则的**水平行**上（底板的"一杠一杠"结构）。

**方法**：
1. 提取红色通道，沿水平方向投影得到**垂直亮度分布**
2. 高斯平滑（σ=3）后做峰值检测，找到所有行位置
3. 对每行提取水平条带（±3 px），沿垂直方向平均后做水平方向峰值检测
4. 对每个检测到的点用**质心法**（center-of-mass）做亚像素位置细化

**结果**：

| 参数 | 值 |
|------|-----|
| 行数 | 84 |
| 行间距 | ~42 px（中值） |
| 每行点数 | 64–85（均值 73） |
| 总发光点 | **6143** |

> v1 的通用局部极大值检测得到 9199 个"点"，其中包含大量行间噪声误检。
> v2 基于行结构检测后减少到 6143 个真实网格节点。

## Step 2: 净荧光信号测量

对每个网格节点：

$$\text{net signal} = \text{spot intensity} - \text{local background}$$

- **spot intensity**: 以节点中心为中心的 7×7 patch 像素均值
- **local background**: 环形区域（内径 r=6, 外径 r=10）像素均值

三张图在**同一组网格节点位置**测量：

| 样品 | 净信号均值 | 净信号 std | 中位数 |
|------|-----------|-----------|--------|
| 0μg/L | 36.9 | 23.0 | 33.7 |
| 5μg/L | 1.4 | 11.0 | −0.4 |
| 20μg/L | −0.0 | 4.5 | −0.4 |

## Step 3: 分区质量过滤

将图像分成 **7×7 = 49 个矩形区块**，对每个区块评估：
- 发光点数量是否足够（≥5 个）
- 背景亮度是否异常偏高（伪影/污染）
- 背景亮度在区块内是否过于不均匀

**结果**：排除 1 个区块（Block(4,2)，背景均值 99.7，明显高于全局中值），保留 **6005 / 6143 个点**（97.8%）。

> 被排除的区块对应图像中一个局部高背景区域（可能是传感器表面污染或杂散光），见 `block_map.png`。

## Step 4: 5 档强度分级

基于 0μg/L 参考图的净信号分布，设定 5 个强度等级：

| Level | 含义 | 净信号范围 |
|-------|------|-----------|
| 0 | 淬灭 (quenched) | ≤ 10.1 |
| 1 | 弱 (dim) | (10.1, 23.4] |
| 2 | 中等 (medium) | (23.4, 40.3] |
| 3 | 较强 (bright) | (40.3, 58.4] |
| 4 | 强 (very bright) | > 58.4 |

淬灭阈值 10.1 由信号噪声水平估计得到（≈ 分布 10th 百分位的绝对值）。其余边界基于有效信号的 90th 百分位范围均匀划分。

### 转换 1: 0μg/L → 5μg/L

$$X = [0.1001,\ 0.2293,\ 0.2713,\ 0.2177,\ 0.1817]$$
$$Y = [0.9116,\ 0.0440,\ 0.0240,\ 0.0130,\ 0.0075]$$

### 转换 2: 0μg/L → 20μg/L

$$X = [0.1001,\ 0.2293,\ 0.2713,\ 0.2177,\ 0.1817]$$
$$Y = [0.9823,\ 0.0135,\ 0.0037,\ 0.0005,\ 0.0000]$$

## 物理解读

- **处理前 (0μg/L)**：发光点强度呈近似正态分布，峰值在"中等"档（27.1%），两侧逐渐下降
- **处理后 (5μg/L)**：91.2% 的点被淬灭至 Level 0，仅 0.75% 保持 Level 4（强亮度）
- **处理后 (20μg/L)**：98.2% 淬灭，无 Level 4 残余，Level 2+ 合计仅 0.42%
- 20μg/L 比 5μg/L 的淬灭更彻底：不仅淬灭比例更高（98.2% vs 91.2%），残余点的亮度也更低

## 与 v1 对比

| 指标 | v1（局部极大值） | v2（网格检测） |
|------|-----------------|---------------|
| 检测方法 | 通用局部极大值 + 阈值 | 行结构 → 行内峰值 |
| 发光点数 | 9199 | 6143（→6005 过滤后） |
| 0μg/L π₀ | 0.012 | 0.100 |
| 伪影过滤 | 无 | 7×7 分区质量评估 |
| 利用先验结构 | 否 | 是（水平行间距 ~42px） |

v1 的 9199 个"点"中约 3000 个是行间噪声误检，导致 π₀（淬灭）被严重低估（1.2% vs 10.0%）。
