TDVP 守恒定律的严格推导：非本征态的演化

2026-04-23 · 回答："为什么 TDVP 不行" · 核心问题："本质上就是一个不在本征态的态的演化"

主线问题：一个不在 Ĥ 本征态的初始态（比如踢后态）用 TDVP 演化，能量 E、归一化 ‖ψ‖²、⟨x²⟩、⟨p²⟩ 理论上应该怎么变？变分流形 M 够不够"装下"真实演化这件事，对守恒定律有没有影响？
主要结论（§2 将严格证明）：
  连续时间 TDVP 下，E 和 ‖ψ‖² 严格守恒，与变分流形是否命中本征态、基是否完备、基是否线性无关都无关。这是 P_T 正交投影的厄米性 + |ψ⟩ ∈ T_ψM 两个事实的纯几何推论。
  ⟨x²⟩、⟨p²⟩ 不是守恒量；对谐振子它们以 角频率 2ω（周期 π）振荡，满足 ⟨x²⟩ + ⟨p²⟩ = 2E。
  数值上 E、norm 飘——不是"不在本征态"导致的，而是 C 矩阵病态 → SVD 伪逆 + Tikhonov → 投影变非厄米 → §2 的证明断了。

1. 记号与 Dirac-Frenkel 变分原理

1.1 变分流形

变分流形 M = { |ψ(z)⟩ : z ∈ ℂⁿ }，z = (u, A, B, R)。切空间

TψM = spanℂ { |∂α ψ⟩ : α = 1, ..., n }

记 PT 为全希尔伯特空间到 TψM 的正交投影算子。显式展开：

PT = |∂α ψ⟩ (C−1)αβ ⟨∂β ψ|,     Cαβ ≡ ⟨∂α ψ | ∂β ψ⟩

关键性质（正交投影）：

PT† = PT,   PT2 = PT. (1)

对 ECG，线性系数 ui 使得 |ψ⟩ = Σi ui |∂ui ψ⟩，所以整体缩放方向一定在切空间内：

|ψ⟩ ∈ TψM   ⟹   PT|ψ⟩ = |ψ⟩. (2)

1.2 TDVP 方程（Dirac-Frenkel）

要求 |ψ̇⟩ ∈ TψM 使残差 ‖ |ψ̇⟩ − (−iĤ)|ψ⟩ ‖² 极小。一阶变分条件 ⟨∂αψ | ψ̇ − (−iĤ)ψ⟩ = 0（∀α）给出

Cαβ żβ = −i gα,    gα = ⟨∂αψ | Ĥ | ψ⟩. (3)

几何上这等价于

|ψ̇⟩ = −i PT Ĥ |ψ⟩. (4)
注：式 (3) 在实现里对应 C·ż = −i·g（见 tdvp_solver.hpp）。当参数化对相位/u-rescale gauge 冗余时，C 有精确零本征方向，但这些方向对 |ψ̇⟩ 的物理贡献为零（因为 PT 只关心切空间像），所以原则上可以用伪逆解决。问题出在"近零"方向——数值上它们被 rcond/λ 截断，见 §6。

2. 守恒定律的严格证明

定理 2.1（能量守恒）：设 Ĥ = Ĥ†，且 |ψ(t)⟩ 满足 TDVP 方程 (4)。则
d/dt ⟨ψ|Ĥ|ψ⟩ = 0. (5)

证明：

d/dt ⟨ψ|Ĥ|ψ⟩ = ⟨ψ̇|Ĥ|ψ⟩ + ⟨ψ|Ĥ|ψ̇⟩ = 2 Re ⟨ψ|Ĥ|ψ̇⟩.

代入 (4)：

⟨ψ|Ĥ|ψ̇⟩ = ⟨ψ| Ĥ · (−i PT Ĥ) |ψ⟩ = −i ⟨ψ| Ĥ PT Ĥ |ψ⟩.

算子 Ĥ PT Ĥ 厄米（Ĥ† = Ĥ，PT† = PT）：

(Ĥ PT Ĥ)† = Ĥ† PT† Ĥ† = Ĥ PT Ĥ.

故 ⟨ψ| Ĥ PT Ĥ |ψ⟩ ∈ ℝ，且

2 Re(−i · ℝ) = 0.   ∎
定理 2.2（归一化守恒）：在 |ψ⟩ ∈ TψM 的前提下（ECG 线性 u 自动满足），TDVP 方程 (4) 给出
d/dt ‖ψ‖² = 0. (6)

证明：

d/dt ⟨ψ|ψ⟩ = 2 Re ⟨ψ|ψ̇⟩ = 2 Re ⟨ψ| (−i PT Ĥ) |ψ⟩ = −2 Im ⟨ψ|PT Ĥ|ψ⟩.

由 PT† = PT 与 (2)：

⟨ψ| PT Ĥ |ψ⟩ = ⟨PT ψ | Ĥ | ψ⟩ = ⟨ψ|Ĥ|ψ⟩ ∈ ℝ,

故 Im(...) = 0。∎

推论：定理 2.1 / 2.2 的两行证明里从未用到：
  • M 的维数 n；
  • {|∂αψ⟩} 是否线性无关（只要用伪逆意义下的 PT 即可，厄米性保留）；
  • |ψ⟩ 是否是 Ĥ 本征态；
  • 真实演化 e−iĤt|ψ(0)⟩ 是否完全落在 M 内。
守恒是 PT 的厄米性 + |ψ⟩ ∈ TψM 的几何事实，仅此而已。

3. TDVP ≠ 真实演化：流形误差

真实演化 |ψ̇⟩exact = −i Ĥ|ψ⟩。投影后的差：

|ψ̇⟩exact − |ψ̇⟩TDVP = −i (1 − PT) Ĥ |ψ⟩. (7)

定义瞬时流形误差

ε(t) ≡ ‖ (1 − PT) Ĥ |ψ(t)⟩ ‖. (8)
情形条件ε(t)TDVP 轨迹
A. ψ 是 Ĥ 本征态 Ĥ|ψ⟩ = E|ψ⟩，且 |ψ⟩ ∈ TψM（ECG 总对） = 0 TDVP = 真实演化（纯相位 e−iEt|ψ⟩）
B. Ĥ|ψ⟩ ∈ TψM 即便 ψ 不是本征态，只要 Ĥ|ψ⟩ 恰落在切空间 = 0 TDVP = 真实演化（局部可积）
C. Ĥ|ψ⟩ ∉ TψM 一般情形 > 0 TDVP ≠ 真实演化，但 E, ‖ψ‖² 仍严格守恒（§2）
关键澄清："不在本征态的态的演化"本身（情形 B 或 C）不是 E 和 norm 飘的来源。它只让 TDVP 的 ψ(t) 和真正的 e−iĤtψ(0) 逐渐错开，体现在 |⟨ψ(0)|ψ(t)⟩|²（fidelity）和 ⟨x²⟩, ⟨p²⟩ 的相位/振幅上——这些不是几何守恒量。

4. ⟨x²⟩, ⟨p²⟩ 在真实演化下的轨迹（Ehrenfest）

取 Ĥ = ½(p̂² + x̂²)（ℏ=m=ω=1）。对易子：

[Ĥ, x̂²] = −i (x̂p̂ + p̂x̂),    [Ĥ, p̂²] = +i (x̂p̂ + p̂x̂).

令 σxx = ⟨x̂²⟩，σpp = ⟨p̂²⟩，σxp = ½⟨x̂p̂ + p̂x̂⟩。Heisenberg 方程 d⟨Â⟩/dt = (1/iℏ)⟨[Â, Ĥ]⟩：

σ̇xx = 2σxp,   σ̇pp = −2σxp,   σ̇xp = σpp − σxx. (9)

记为 ṡ = M s，其中

M = [[0, 0, 2], [0, 0, −2], [−1, 1, 0]]

特征多项式 det(M − λI) = −λ(λ² + 4)，本征值 {0, ±2i}。

  • 零模：对应守恒量 σxx + σpp = 2E（HO virial 的动力学版）。
  • ±2i 模：频率 2ω = 2，周期 T = π。

闭式解（D0 = σxx(0) − σpp(0)，V0 = σxp(0)）：

σxx(t) = E + (D0/2) cos(2t) + V0 sin(2t)
σpp(t) = E − (D0/2) cos(2t) − V0 sin(2t)
σxp(t) = −(D0/2) sin(2t) + V0 cos(2t) (10)

4.1 代入踢后态（κ=1, kL=0.5）

踢前基态 ψ0(x) = π−1/4 e−x²/2。kick 算符 Û = e−iκ cos(2kLx) 只改相位，不改 |ψ|²。定义 φ(x) = κ cos(2kLx)。踢后态 ψ' = e−iφ ψ0。

σxx(0) = ⟨x²⟩ψ0 = 1/2

对 ⟨p²⟩，用 p̂ e−iφ ψ0 = e−iφ(p̂ + ℏφ'(x))ψ0：

σpp(0) = ⟨(p̂ + φ'(x))²⟩ψ0 = ⟨p̂²⟩0 + 2⟨p̂φ'⟩sym + ⟨φ'²⟩0

对实基态 ⟨p̂φ' + φ'p̂⟩0 = 0；φ'(x) = −2κkL sin(2kLx)：

⟨φ'²⟩0 = 4κ²kL² ⟨sin²(2kLx)⟩0 = 2κ²kL² (1 − e−4kL²)

代入：σpp(0) = 1/2 + 2κ²kL²(1 − e−4kL²) ≈ 0.8161。

对 σxp(0)：利用 ⟨ψ'|x̂p̂|ψ'⟩ = ⟨ψ0| x̂(p̂ + φ') |ψ0⟩，实部即：

V0 = σxp(0) = ⟨x̂ φ'(x)⟩0 = −2κkL ⟨x sin(2kLx)⟩0

用 ⟨x sin(ax)⟩0 = (a/2) e−a²/4（标准高斯积分），令 a = 2kL：

⟨x sin(2kLx)⟩0 = kL e−kL²,    V0 = −2κ kL² e−kL²

对 κ=1, kL=0.5：V0 ≈ −0.5 · e−0.25 ≈ −0.389。

符号修正：上一轮对话里我写的是 V0 = +2κkL²e−kL²，这里严格化后 V0 ≈ −0.389（来自 φ' 的负号）。振幅不变，只是相位差 π。

5. 回到"不在本征态"这件事

用户的原话："本质上就是一个不在本征态的态的演化。"

在 TDVP 的语境下，"不在本征态"可以解析成两件不等价的事：

含义ε(t)后果
(a) |ψ⟩ 不是 Ĥ 本征态：Ĥ|ψ⟩ 与 |ψ⟩ 不平行 可能为 0 也可能 > 0 —
(b) Ĥ|ψ⟩ 不在 TψM 内（流形不够大） > 0 TDVP 轨迹离开真实轨迹，但 E、norm 仍精确守恒

踢后态同时满足 (a) 和 (b)：

  • (a) 是物理事实：kick 激发了高能模式。
  • (b) 取决于变分基。K=1 ECG 只能写一个 Gaussian，绝无法表达 e−iκ cos(2kLx) ψ0——必须 augment 出动量副本（task4 Step 3.5 做的事）。加得够多，(b) 的残差能压低，但永远 ε > 0。
总结：(a) + (b) 让 TDVP 轨迹偏离真实 Schrödinger 演化（表现在 |F(t)|² < 1、σxx 相位漂移），但不会让 E 或 ‖ψ‖² 漂。这两个守恒是由式 (4) 的几何结构保证的，与 ε 是否为零无关。

6. 数值上为什么还是飘？

§2 的证明关键用到 (1)：PT† = PT。这个等式需要 C 精确可逆且 C−1 被完整保留。数值现实：

6.1 Gauge / 过参数化 → C 病态

ECG 参数化 |ψ⟩ = Σ ui φi，多个 φi 可以通过小调 (u, A, B, R) 互相吸收：

  • 相位 gauge：ui → eiα ui（全局相位）是 C 的精确零本征方向。
  • u-rescale gauge：(ui, 其他) → (c·ui, 对应调整)，也是零方向。
  • augment 出的动量副本共线：n=+1 和 n=−1 副本有几乎 mirror 的 ⟨φi|φj⟩，cond(C) 1e10+（rothe_implementation_plan.html 已诊断）。

6.2 SVD 伪逆 + Tikhonov → 投影变非厄米

实现里 C−1 被替换为

C̃−1 = (C + λI)−1 截断小于 rcond·σmax 的 SVD 模式

对应的"投影算子"

P̃T = |∂α⟩ (C̃−1)αβ ⟨∂β|

一般不再满足 P̃T† = P̃T（Tikhonov 加的 λI 保厄米，但 rcond 截断的"伪逆"对左右奇异向量不对称处理时就破对称）。即便精确保留厄米性，P̃T ≠ P̃T2，它不再是投影算子。§2 的证明关键用 "PT 自伴 → Ĥ PT Ĥ 自伴" 这一步断了：

(Ĥ P̃T Ĥ)† = Ĥ P̃T† Ĥ ≠ Ĥ P̃T Ĥ   ⟹   ⟨ψ|Ĥ P̃T Ĥ|ψ⟩ ∉ ℝ

其虚部贡献 Re(−i · 复数) 就是能量漂的瞬时速率。

6.3 离散积分

即使 PT 精确，RK4 用在 ż = −i C−1 g 上有 O(dt4) 能量漂；但 RK4 不是 symplectic 积分器（即 ż = f(z) 形式没有辛结构），所以漂是系统偏置的。

6.4 为什么 K=1 基态（task6）应该是机器精度

K=1 基态：Ĥ|ψ⟩ = ½|ψ⟩ = (½/u) |∂uψ⟩ ∈ TψM。

  • ε = 0（情形 A + B）；
  • C 是 4×4（u, A, B, R）——有 u-相位 gauge 方向精确零，其他方向良定；
  • 即便伪逆截掉零方向，|ψ̇⟩ = −iE|ψ⟩ 的物理解依然落在非零方向上，不受影响；
  • ż 应该只有 u̇ = −iE u，其他方向精确 0；
  • 离散误差 O(dt4) 对 RK4 只影响 u 的相位精度——不影响 E, ‖ψ‖², ⟨x²⟩, ⟨p²⟩。
所以：如果 task6 跑出 max|ΔE| > 1e-8，或 A/B/R 的 dz 非零，那不是 gauge 问题，而是装配的 bug——比如 g_α 里 H 矩阵元的构造、C 矩阵的装配、Wirtinger 导数约定之类。切 Rothe 前这个必须先过。

7. 数值验证：双 kick + 有限脉冲 TDVP (task7)

目的：把前面 §2–§6 的理论判断用 Route-1 踢后态 + 自由演化 + 第二次 kick 脉冲 的设置打通一遍，看实测数字和理论判断是否对得上。

7.1 设置

  1. 起点：Route-1 踢后态 |ψ(0)⟩ = Ukick ψ0，在 Kpre=1 + nmom=2 augment 出的基底上拟合，Kcap 截到 3 或 5。
  2. 阶段 A (t ∈ [0, Tfree])：TDVP 实时演化，Ĥ = T̂ + V̂ho。
  3. 阶段 B (t ∈ [Tfree, Tfree + Tpulse])：TDVP 实时演化，Ĥpulse = (κ/Tpulse)cos(2kLx)，使 ∫Ĥpulse dt = κcos(2kLx) 还原瞬时 kick 极限。
关键理论观察（简化脉冲期）：Ĥpulse 是纯位置算符，任意时刻对易 [Ĥpulse(t1), Ĥpulse(t2)] = 0。故
Upulse = exp(-i ∫0Tpulse Ĥpulse dt) = exp(-iκcos(2kLx)) = U瞬时 kick
与 Tpulse 的值无关——真实演化 Schrödinger 角度 Tpulse = 0.01, 0.1, 1, 100 都给出相同末态。所以 Tpulse 的选择纯粹是 TDVP 的数值稳定性参数，不是物理参数。

7.4.2 核心反差：ΔE 没改善

从 12 → 9 → 6 参数，ΔE 是 0.33 → 0.67 → 0.64——不跟着 gauge 消除单调降低。这说明 E 漂的主导机制不是 §6.2 的"投影非厄米"，而是 §3 的流形误差 ε：

ε(t) = ‖(1-PT)Ĥpulse|ψ(t)⟩‖

直接看数字：网格给 E(kick 2) = 0.853，TDVP 不论 K=3 基怎么配都落在 1.3 附近——流形再怎么内部"干净"，装不下 e-iκcos(2kLx) ψ(Tfree) 的 Jacobi-Anger 高阶展开（|n| ≥ 3）。ε 的能量投影分量就直接是 E 漂的"来源"。

这是 §3 流形误差 ε 的实测体现：
  • §2 守恒证明只用 PT 厄米性 → E 能"守恒到错的值"。6 参数配置 max|ΔE|=0.64 看起来是"不守恒"，但注意 Hpulse 包含 Vkick，TDVP 测的 E = ⟨T+Vho⟩ 不是脉冲期的守恒量；脉冲 H 的守恒量是 ⟨Ĥpulse⟩，TDVP 对它仍守恒（sv 未打底），但这不约束 E = ⟨T+Vho⟩。
  • 真正问题：TDVP 轨迹 |ψTDVP(t)⟩ 本身偏离真实 e-iĤpulset|ψ(Tfree)⟩，体现在测出来的 ⟨T+Vho⟩ 和网格差 52%。
  • 不是 gauge、不是伪逆、不是离散误差——是流形本身不够大。
