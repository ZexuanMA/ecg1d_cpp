1. 物理目标与 MBDL 背景

长期目标：量化模拟 Guo et al. Science 389, 716 (2025) 的多体动力学局域化（MBDL）。原文实验观测三大信号：

  1. 动量分布 n(k) 冻结
  2. 能量和熵的饱和
  3. 一阶关联函数 G(1)(z) 的特征衰减

文章的理论建模（QMC + Floquet）只捕捉到定性一致。ECG + TDVP 提供了从变分动力学角度做定量预测的可能。

第一步（本阶段）：在 N=1 + 单粒子谐振子 + 周期 kick 的最简 Hamiltonian 上打通方法：

Ĥ(t) = T̂ + V̂ho + Vkick(x) · Σn δ(t − nT)

单周期演化 U = Ukick · Ufree(T)。N=1 没有相互作用也不是严格的 MBDL，但为多体情形打 TDVP 基础。

验收（task5）：实时 TDVP 在 T=2π 一个谐振周期内

  • 撑完 628 步 @ dt=1e-2（或 6283 步 @ dt=1e-3）不崩
  • E(t) 守恒（H 不含 t → 理论严格守恒）
  • ‖ψ‖² 守恒（幺正）
  • |⟨ψ(0)|ψ(T)⟩|² ≈ 1（HO 谱 En = n+½ 的 recurrence）

2. 理论基础：TDVP 守恒定律

2.1 变分流形与 Dirac-Frenkel 原理

变分流形 M = {|ψ(z)⟩ : z ∈ ℂⁿ}，z = (u, A, B, R)；切空间 TψM = span { |∂αψ⟩ }；PT 为正交投影到切空间（厄米、幂等）。ECG 的线性 u 使 |ψ⟩ ∈ TψM。

Dirac-Frenkel 最小化 ‖|ψ̇⟩ − (−iĤ)|ψ⟩‖² in TψM，给出

Cαβ żβ = −i gα,   Cαβ = ⟨∂αψ|∂βψ⟩,   gα = ⟨∂αψ|Ĥ|ψ⟩ (1)

几何上：|ψ̇⟩ = −i PT Ĥ |ψ⟩。

2.2 严格守恒定理

定理：Ĥ = Ĥ† + TDVP 方程 (1) + |ψ⟩ ∈ TψM ⟹
d⟨ψ|Ĥ|ψ⟩/dt = 0,   d‖ψ‖²/dt = 0 (2)

证明（两行）：

d⟨ψ|Ĥ|ψ⟩/dt = 2Re⟨ψ|Ĥ|ψ̇⟩ = −2Re(i⟨ψ|Ĥ PT Ĥ|ψ⟩)

注意 Ĥ PT Ĥ 厄米（Ĥ†=Ĥ，PT†=PT），期望值实数，Re(−i·ℝ) = 0。∎

推论：证明不依赖 M 的维数、基的线性无关、ψ 是否本征态、流形是否装下真实演化。只依赖 PT 的厄米性。这是纯几何事实。

2.3 流形误差 ε(t)

真实演化 |ψ̇⟩exact = −iĤ|ψ⟩，和 TDVP 的差：

ε(t) = ‖(1 − PT) Ĥ|ψ(t)⟩‖ (3)
情形条件ε(t)TDVP 轨迹
A. ψ 是 Ĥ 本征态Ĥ|ψ⟩ ∝ |ψ⟩ ∈ TψM= 0TDVP = 真实演化（纯相位）
B. 非本征态，但 Ĥ|ψ⟩ ∈ TψM流形够大= 0TDVP = 真实演化
C. Ĥ|ψ⟩ ∉ TψM一般情形> 0TDVP ≠ 真实演化，但 E、‖ψ‖² 仍守恒

2.4 Ehrenfest：⟨x²⟩, ⟨p²⟩ 在真实演化下

对 Ĥ = ½(p̂² + x̂²)，令 σxx = ⟨x̂²⟩ 等：

σ̇xx = 2σxp,  σ̇pp = −2σxp,  σ̇xp = σpp − σxx

本征值 {0, ±2i}。周期 π 振荡 + 守恒量 σxx + σpp = 2E。

对 κ=1, kL=0.5 踢后态：E ≈ 0.658, D0 ≈ −0.316, V0 = −2κkL²e−kL² ≈ −0.389，振幅 ≈ 0.42。

3. 数值上为什么还是飘

§2 的证明用到 PT† = PT。实现里这一步断：

3.1 Gauge / 过参数化 → C 病态

  • 相位 gauge：ui → eiα ui 是 C 的精确零本征方向
  • 动量副本共线：augment 的 n=±1 几近 mirror → cond(C) 1e10+
  • Width 重用：greedy SVM 选相同 width 多份 → ∂A, ∂B 平行

3.2 SVD 伪逆 + Tikhonov → 投影非厄米

实际用 C̃−1 = (C + λI)−1 截断 sv < rcond·σmax 后的 SVD 模式。对应 P̃T：

(Ĥ P̃T Ĥ)† ≠ Ĥ P̃T Ĥ ⟹ ⟨ψ|Ĥ P̃T Ĥ|ψ⟩ ∉ ℝ

虚部贡献 Re(−i·复数) ≠ 0 → 能量漂速。

3.3 离散积分

RK4 on ż = −iC−1g 有 O(dt⁴) 能量漂；但不辛 → 偏置不对消。

3.4 为什么 K=1 基态（task6）是机器精度

  • Ĥ|ψ⟩ = ½|ψ⟩ ∈ TψM → ε=0（情形 A）
  • C 4×4，伪逆截掉 u-相位零方向不影响物理解
  • ż 只有 u̇ = −iE u，A/B/R 精确 0

实测：628 步全程机器零误差。核心装配无 bug，所有 K≥3 的失败归因到 §3.1-3.2。

4.3 当前成功点：Wiener + 小 dt

Wiener 平滑 sinv = sv / (sv² + λ²)：

  • sv=0 严格给 0 → 真 gauge 方向精确清零（必须 λC=0）
  • sv 连续函数 → 无阶梯跳（克制模式 C）
  • 有限放大 1/(2λ) → 比硬 rcond 温和

代价是有偏差（不是 TDVP 严格解），每步 O(λ²/sv)，积累与 dt 成正比。所以 dt 必须小：

  • dt=1e-2：bias 推到错位 manifold，E 锁在 5.96 失败
  • dt=1e-3：bias 可控，E 恒定 4000+ 步到 6 位小数 成功

5. 实验 · task7：双 kick + 有限脉冲

动机：task5 单 kick 后的实时 TDVP 能撑，但没告诉我们 K 基底能否支持第二次 kick。task7 直接做：踢一次 → 自由演化 Tfree → 再踢一次，和网格精确比。

5.1 设置

  1. 起点：Route-1 踢后态 |ψ(0)⟩ = Ukick ψ0
  2. 阶段 A (t ∈ [0, Tfree])：TDVP 实时演化，Ĥ = T̂ + V̂ho
  3. 阶段 B (t ∈ [Tfree, Tfree+Tpulse])：TDVP 实时演化，Ĥpulse = (κ/Tpulse)cos(2kLx)，∫Ĥpulsedt = κcos(2kLx) 还原瞬时 kick
关键观察（脉冲）：Ĥpulse 是纯位置算符，[Ĥpulse(t1), Ĥpulse(t2)] = 0。严格演化
Upulse = exp(−i∫0Tpulse Ĥpulse dt) = exp(−iκcos(2kLx)) = U瞬时 kick
与 Tpulse 无关——真实演化 Tpulse = 0.01, 0.1, 1, 100 都给相同末态。所以 Tpulse 纯粹是 TDVP 的数值稳定性参数。

5.4.1 单调 trend（符合 §3.2 理论）

12 → 9 → 6 参数对应依次消除：(i) B 带的 3 个冗余方向，(ii) u 的全局相位 gauge。svmin3 的 1e-8 底从 3 个降到 0 个，Δ⟨x²⟩ 单调从 0.089 降到 0.026。

5.4.2 核心反差：ΔE 不跟 gauge 消除降低

ΔE 是 0.33 → 0.67 → 0.64，不单调。网格 E=0.853，TDVP 不论 K=3 怎么配都落在 1.3 附近。流形装不下 Jacobi-Anger 高阶项（|n| ≥ 3）→ ε 主导。

这是 §2.3 情形 C 的实测：
  • 6 参数配置 sv 清底 → §2 守恒证明没断 → TDVP 对脉冲期的 ⟨Ĥpulse⟩仍守恒
  • 但测的 E = ⟨T+Vho⟩ ≠ 脉冲期守恒量；TDVP 的 ψ(t) 本身偏离真实 e−iĤpulset|ψ(Tfree)⟩
  • 结果：Δ⟨x²⟩ 可以通过消 gauge 降低（流形内几何问题），但 E(末) 离网格的距离是 ε 的函数，K 固定时无法进一步缩

6.2 判决：K 固定的 TDVP 是死路

task7 的技术判决：
  1. 阶段 A（Ĥ|ψ⟩ 几乎在 TψM）：消完 gauge 后 TDVP 可用（K=3 ΔE=1.6e-5）
  2. 阶段 B（kick 把态扔出流形）：TDVP 几何守恒不破，但 ψ(t) 已和真实演化脱钩。K 固定无法修
  3. 物理上第二次 kick 把态的 Jacobi-Anger 展开推到更高阶（|n|≥3），K=3 的 ±1,±2 动量副本完全装不下

6.3 下一步：Rothe 方法

Schrader-Pedersen-Kvaal JCP 2025 方法：每步最小化 Crank-Nicolson 残差

|ψ(t+dt)⟩ = arg minχ ‖ Â|χ⟩ − Â†|ψ(t)⟩ ‖²,  Â = 1 + i(dt/2)Ĥ

VarPro 给出线性系数 c = S̃−1ρ̃ 解析，α 留给 BFGS。不反 C，改反 overlap 类 S̃（总是良定）。自适应加基/删基：残差超阈值就加一个 Gaussian，可以把 ε 压到可控上界下。

关键优势：

  • Crank-Nicolson 无条件稳定（A-stable）
  • 累积 Rothe 残差是真实演化误差的显式上界（paper 式 17-19）
  • 2D Hénon-Heiles 实验：30 Gaussian 做到定量结果
