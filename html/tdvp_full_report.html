已做到的：

  • K=1 HO 基态实时 TDVP 自检：628 步全程机器零误差 ✓（证明核心装配无 bug）

  2. 流形层：K 固定时流形装不下 kick 后的 Jacobi-Anger 高阶展开 ⇒ TDVP 轨迹偏离真实 Schrödinger 演化，E(kick 2) 落在 1.30 vs 网格精确 0.853（偏 52%）

  4. 结论：K 固定的 TDVP 无法修。唯一出路是 Rothe 方法（JCP 2025）+ 自适应加基，已在 task8 完成 M1+M2。

下一步：推进 Rothe M3（K=2–3 踢后态演化），对比网格精确值，若对上则切换为主线方法并继续 MBDL 多 kick。

1. 物理目标与 MBDL 背景

长期目标：量化模拟 Guo et al. Science 389, 716 (2025) 的多体动力学局域化（MBDL）。原文实验观测三大信号：

  1. 动量分布 n(k) 冻结
  2. 能量和熵的饱和
  3. 一阶关联函数 G(1)(z) 的特征衰减

文章的理论建模（QMC + Floquet）只捕捉到定性一致。ECG + TDVP 提供了从变分动力学角度做定量预测的可能。

第一步（本阶段）：在 N=1 + 单粒子谐振子 + 周期 kick 的最简 Hamiltonian 上打通方法：

Ĥ(t) = T̂ + V̂ho + Vkick(x) · Σn δ(t − nT)

单周期演化 U = Ukick · Ufree(T)。N=1 没有相互作用也不是严格的 MBDL，但为多体情形打 TDVP 基础。

验收（task5）：实时 TDVP 在 T=2π 一个谐振周期内

  • 撑完 628 步 @ dt=1e-2（或 6283 步 @ dt=1e-3）不崩
  • E(t) 守恒（H 不含 t → 理论严格守恒）
  • ‖ψ‖² 守恒（幺正）
  • |⟨ψ(0)|ψ(T)⟩|² ≈ 1（HO 谱 En = n+½ 的 recurrence）

2. 理论基础：TDVP 守恒定律

2.1 变分流形与 Dirac-Frenkel 原理

变分流形 M = {|ψ(z)⟩ : z ∈ ℂⁿ}，z = (u, A, B, R)；切空间 TψM = span { |∂αψ⟩ }；PT 为正交投影到切空间（厄米、幂等）。ECG 的线性 u 使 |ψ⟩ ∈ TψM。

Dirac-Frenkel 最小化 ‖|ψ̇⟩ − (−iĤ)|ψ⟩‖² in TψM，给出

Cαβ żβ = −i gα,   Cαβ = ⟨∂αψ|∂βψ⟩,   gα = ⟨∂αψ|Ĥ|ψ⟩ (1)

几何上：|ψ̇⟩ = −i PT Ĥ |ψ⟩。

2.2 严格守恒定理

定理：Ĥ = Ĥ† + TDVP 方程 (1) + |ψ⟩ ∈ TψM ⟹
d⟨ψ|Ĥ|ψ⟩/dt = 0,   d‖ψ‖²/dt = 0 (2)

证明（两行）：

d⟨ψ|Ĥ|ψ⟩/dt = 2Re⟨ψ|Ĥ|ψ̇⟩ = −2Re(i⟨ψ|Ĥ PT Ĥ|ψ⟩)

注意 Ĥ PT Ĥ 厄米（Ĥ†=Ĥ，PT†=PT），期望值实数，Re(−i·ℝ) = 0。∎

推论：证明不依赖 M 的维数、基的线性无关、ψ 是否本征态、流形是否装下真实演化。只依赖 PT 的厄米性。这是纯几何事实。

2.3 流形误差 ε(t)

真实演化 |ψ̇⟩exact = −iĤ|ψ⟩，和 TDVP 的差：

ε(t) = ‖(1 − PT) Ĥ|ψ(t)⟩‖ (3)
情形条件ε(t)TDVP 轨迹
A. ψ 是 Ĥ 本征态Ĥ|ψ⟩ ∝ |ψ⟩ ∈ TψM= 0TDVP = 真实演化（纯相位）
B. 非本征态，但 Ĥ|ψ⟩ ∈ TψM流形够大= 0TDVP = 真实演化
C. Ĥ|ψ⟩ ∉ TψM一般情形> 0TDVP ≠ 真实演化，但 E、‖ψ‖² 仍守恒

2.4 Ehrenfest：⟨x²⟩, ⟨p²⟩ 在真实演化下

对 Ĥ = ½(p̂² + x̂²)，令 σxx = ⟨x̂²⟩ 等：

σ̇xx = 2σxp,  σ̇pp = −2σxp,  σ̇xp = σpp − σxx

本征值 {0, ±2i}。周期 π 振荡 + 守恒量 σxx + σpp = 2E。

对 κ=1, kL=0.5 踢后态：E ≈ 0.658, D0 ≈ −0.316, V0 = −2κkL²e−kL² ≈ −0.389，振幅 ≈ 0.42。

3. 数值上为什么还是飘

§2 的证明用到 PT† = PT。实现里这一步断：

3.1 Gauge / 过参数化 → C 病态

  • 相位 gauge：ui → eiα ui 是 C 的精确零本征方向
  • 动量副本共线：augment 的 n=±1 几近 mirror → cond(C) 1e10+
  • Width 重用：greedy SVM 选相同 width 多份 → ∂A, ∂B 平行

3.2 SVD 伪逆 + Tikhonov → 投影非厄米

实际用 C̃−1 = (C + λI)−1 截断 sv < rcond·σmax 后的 SVD 模式。对应 P̃T：

(Ĥ P̃T Ĥ)† ≠ Ĥ P̃T Ĥ ⟹ ⟨ψ|Ĥ P̃T Ĥ|ψ⟩ ∉ ℝ

虚部贡献 Re(−i·复数) ≠ 0 → 能量漂速。

3.3 离散积分

RK4 on ż = −iC−1g 有 O(dt⁴) 能量漂；但不辛 → 偏置不对消。

3.4 为什么 K=1 基态（task6）是机器精度

  • Ĥ|ψ⟩ = ½|ψ⟩ ∈ TψM → ε=0（情形 A）
  • C 4×4，伪逆截掉 u-相位零方向不影响物理解
  • ż 只有 u̇ = −iE u，A/B/R 精确 0

实测：628 步全程机器零误差。核心装配无 bug，所有 K≥3 的失败归因到 §3.1-3.2。

4. 起点态的准备：Route-1 踢后态构造

task5 和 task7 的起点都是 ψ0 被踢一次后的 ECG 表示。这里详细说明怎么得到、和 K 取值的影响。

4.1 为什么不能直接一步算出

瞬时 kick 算符 Ukick = e−iκcos(2kLx) 作用在 HO 基态上：

Ukick ψ0(x) = e−iκcos(2kLx) · π−1/4 e−x²/2

这不是 Gaussian——cos(2kLx) 相位让波函数有周期性调制。ECG 单 Gaussian 基 K=1 绝无法表达。

把 kick 算符展成 Jacobi-Anger：

e−iκcos(2kLx) = Σn=−∞∞ (−i)n Jn(κ) ei·n·2kLx (4)

每一项 ei·n·2kLx ψ0(x) 仍是 Gaussian，只是中心从 R=0 平移到 R=i·n·kL/B（虚 R 编码动量）。所以

Ukick ψ0 = Σn (−i)n Jn(κ) · φn(x),    φn(x) 是动量 n·2kL 的 Gaussian (5)

要在 ECG 基里装下，必须扩基到 多个动量副本。

4.2 Route-1 target-fitting TDVP（task4）

工作流（见 src/target_fitting.{hpp,cpp}、src/kick_operator.cpp）：

  1. 构造目标态 |ψtar⟩：按 (5) 截断到 |n| ≤ nmax（默认 nmax=4），每项作为一个显式 Gaussian。这是"精确答案"，但需要 2nmax+1 = 9 个基。
  2. 扩基 trial basis（augment_basis_with_momentum）：对 pre-kick 基底 Kpre，每个 Gaussian 复制出 n = ±1, ±2, ..., ±nmom 的动量副本。内部维护一个候选池（4 种宽度 × 动量对 = 16 个候选），按 cond(S)<max_cond 过滤，取前 Kcap 个。
  3. VarPro 解析最优 u：对固定的非线性参数 (A, R)，
uopt = Strial−1 · ρ,    ρm = ⟨φmtrial | ψtar⟩
  1. TDVP 迭代（可选）：对非线性参数 (A, R) 用 target-fit TDVP 梯度下降最小化 1−F，F 为 fidelity：
    F = |⟨ψtar|ψtrial⟩|² / (⟨ψtar|ψtar⟩ · ⟨ψtrial|ψtrial⟩)
    实测 VarPro 给出的初值 F 已经很接近线性最优，TDVP 挪非线性参数常常 1-2 步就 line search fail（improvement 已经饱和）。

4.5 一个合理质疑：为什么不直接把 target 当基底？

读者常问："Jacobi-Anger 截断的每一项 φn = ei·n·2kLxψ0 本身就是 Gaussian，就是 ECG 基——为什么要走 trial → target 拟合那一套？直接用 target 当基底不就 F=1 了吗？"

答：完全正确。φn 在 ECG 约定下的参数：

φn(x) = exp(−(A+B) x² + 2RBx − R²B),   A+B = 0.5,  Rn = i·n·kL/B

把 (5) 的 9 个 φn（|n|≤4）直接当基底，令 un = (−i)n Jn(κ)，F = 1 严格成立（除 Jacobi-Anger 截断误差，但这和 Route-1 相同，不是基表达能力问题）。

那 Route-1 在做什么

Route-1 的 target-fitting 不是"把 target 表达成 ECG"——target 本身已经是 ECG。它是"把另一组多宽度的 trial 基底拟合到 target"：

基底构成存在目的
Target9 Gaussian, 单宽度 A+B=0.5, n=−4..+4精确表达 t=0 的踢后态
Trial（Route-1）4 宽度 × n=±1,±2, + 原件 = 9给下游演化留适应性

实测 1−F = 6e-4 不是"ECG 装不下 target"，而是"trial 的张成空间 ≠ target 的张成空间"——两组 9 个 Gaussian 都是 ECG，但张不同子空间。

物理假设

关键事实：φn = ei·n·2kLxψ0 是 HO 的相干态（动量平移的基态）。HO 里相干态时间演化宽度恒定，只有 (R, p) 做经典正弦振荡。看上去 target 基在阶段 A 应该足够（不需要多宽度），甚至更稳（不浪费 gauge 冗余方向）。

原假设 H1（基于"相干态在 HO 不变宽"）：target 基（单宽度, 9 Gaussian）比 Route-1 trial 基（多宽度）在阶段 A 稳定性更好——max|ΔE|、cond(C) 都应改善。

为什么反转？—— 参数数量 > 基的"物理对齐质量"

原假设只考虑了"基底表达的物理对齐"（单宽度 = HO 相干态自然形状），忽略了 TDVP 的瓶颈其实是 C 矩阵的 gauge 冗余方向数量。

  • task7 K=3 trial（6 参数）：基少，cond(C) 最小，ΔE=1.6e-5 🏆
  • task9 K=5 target（10 参数）：基的质量完美（F=1），但 10 参数 ≫ 6，阶段 A ΔE 漂 5 位数
  • task9 K=9 target（18 参数）：3 个 sv 直接打 λC=1e-8 底，单步 |dz|=258，立刻崩

把 §3.1 的"gauge 冗余 → C 病态"重新量化：

阶段 A max|ΔE| 大致 ∝ (参数数)p,   p ≳ 4 实测

即便 target 基 φn 在物理上"只差动量相位"，数值上 9 个 ei·n·2kLxψ0 之间 Gram 矩阵是 Snm = √π·e−(n−m)²·kL²，高斯核在 9 个格点上产生多个非常小的奇异值——对应"相邻 n 的线性组合 ≈ 中间 n"。这些是 C 的近零方向，rcond/λ 截断它们就是 §3.2 的"投影非厄米"重现。

4.6 对下游实验（task5, task7）的意义

Route-1 给的初态都有 ε > 0（基底没完全覆盖真实态，§2.3 情形 C）。但阶段 A 自由演化期间，这个初态的能谱分量在 HO 本征基下稳定，⟨T+Vho⟩ 在切空间内演化得不错：

  • task5（T=2π 实时 TDVP）：K=3 初态 F=0.989，Wiener+dt=1e-3 撑到 |F(T)|² = 0.874，ΔE=11%
  • task7 阶段 A（T=1 自由演化）：K=3 ΔE = 1.6e-5，K=5 ΔE = 1.2e-3——初态 fidelity 的残差在自由演化里不会被放大
  • task7 阶段 B（第二次 kick）：这里瓶颈显现——第二次 kick 把已经不完美的 Jacobi-Anger 展开再卷一次，等效于需要 |n|≤8 的展开；K=3 的 ±1,±2 完全装不下 → ε 放大 → TDVP 轨迹偏离真实演化，E 落在 1.30 vs 网格 0.85

这正是 §6 为什么推 Rothe 的自适应加基：残差超阈值就 往基里加新 Gaussian，让流形动态跟上物理态的复杂度。

  • sv 连续函数 → 无阶梯跳（克制模式 C）
  • 有限放大 1/(2λ) → 比硬 rcond 温和

代价是有偏差（不是 TDVP 严格解），每步 O(λ²/sv)，积累与 dt 成正比。所以 dt 必须小：

  • dt=1e-2：bias 推到错位 manifold，E 锁在 5.96 失败
  • dt=1e-3：bias 可控，E 恒定 4000+ 步到 6 位小数 成功

7.2 判决：K 固定的 TDVP 是死路

task7 的技术判决：
  1. 阶段 A（Ĥ|ψ⟩ 几乎在 TψM）：消完 gauge 后 TDVP 可用（K=3 ΔE=1.6e-5）
  2. 阶段 B（kick 把态扔出流形）：TDVP 几何守恒不破，但 ψ(t) 已和真实演化脱钩。K 固定无法修
  3. 物理上第二次 kick 把态的 Jacobi-Anger 展开推到更高阶（|n|≥3），K=3 的 ±1,±2 动量副本完全装不下
