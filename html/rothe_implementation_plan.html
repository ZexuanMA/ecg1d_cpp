Rothe 方法实施方案（ECG 实时演化替代 TDVP）

生成于 2026-04-22 · 切换动机、数学核心、复用代码、H² 矩阵元、里程碑

为什么要切： vMCG 风格 TDVP 实时演化在我们过参数化的 K=10 基下固有不稳定（Gram cond 1e10+，任意 rcond/λ 组合都会在某个 t 爆炸）。Richings 2015 Int. Rev. Phys. Chem. review 原话："When the basis set is complete, C becomes zero. The vMCG EOMs become very unstable." 在 Tikhonov 上继续调不是路。Rothe 方法（Schrader-Pedersen-Kvaal, JCP 2025）根本不反 C，而是把每个时间步变成一个非线性最小二乘问题。已被证明可在 2D Hénon-Heiles 上 30 个 Gaussian 跑出定量结果。

1. 数学核心（三步 + VarPro）

Step 1 — Crank-Nicolson 传播子

TDSE i ∂t|ψ⟩ = H|ψ⟩（ℏ=1）。CN 传播子（paper 式 5-6）：

|ψ(t+dt)⟩ = Âi−1 Âi† |ψ(t)⟩,   Âi = 1 + i(dt/2) Ĥ(ti+dt/2)

优点：对厄米 H 严格幺正、无条件稳定（A-stable）、二阶精度。缺点：Â−1 是无穷维算子，一般变分基下没法直接算。

Step 2 — 换成最小化（Rothe's trick）

Âi |ψ(t+dt)⟩ = Âi† |ψ(t)⟩ 等价于：

|ψ(t+dt)⟩ = arg minχ ‖ Âi |χ⟩ − Âi† |ψ(t)⟩ ‖²

这个残差 r 叫 Rothe error。核心性质（paper 式 17）：

‖ |ψ(t+dt)⟩ − Âi−1 Âi† |ψ(t)⟩ ‖² ≤ ropt

→ 每步的 Rothe error 是 CN 真实演化误差的上界；累积 Rothe error 是总误差的上界（paper 式 18-19）。这是 TDVP 不具备的可控性——TDVP 只有 O(dt⁴) 抽象承诺，没有可计算的误差估计。

Step 3 — ansatz & VarPro（Variable Projection）

变分 ansatz（paper 式 8）：|χ⟩ = Σm cm |φm(α)⟩。α 是非线性参数向量（我们的 A, B, R），c 是线性系数（我们的 u）。

VarPro 的魔法（Golub-Pereyra 1973，paper 式 12-14）：给定任意 α，最优 c 可以解析求——就是个普通的线性最小二乘：

c(α) = S(α)−1 ρ(α),    Smn(α) = ⟨Â φm(α) | Â φn(α)⟩,   ρm(α) = ⟨Â φm(α) | Â† ψ(t)⟩
关键对比： 这里的 S(α) 是普通 overlap-like 矩阵（Gram 矩阵），不是我们那个病态 TDVP 度量 C。S 几乎总是正定且良好条件。Schrader 2025 实践中只在 t=0 附近 2D 情况下才加 λ=1e-8 Tikhonov 到 S（paper 式 51），其余时刻裸反就行。

代回，残差只剩 α 的函数（paper 式 11）：

r(α) = ‖ Σm cm(α) Â φm(α) − Â† ψ(t) ‖²

Step 4 — 对 α 做 BFGS 最小化（paper §II.D）：解析梯度 ∇α r(α)，BFGS 迭代，几十次收敛。初值用前两步外推 αinit = αi + δ(αi − αi−1)，δ 由线搜索选。

3. 与 Task D（Route 1 target fitting）的亲缘关系

核心观察：Rothe 方法 = Task D 内嵌在外层时间循环。

Task D 在做的事：给定 |target⟩，找 ECG 基 {α, c} 使得 ‖ |target⟩ − |χ⟩ ‖² 最小（严格说用的是 1−fidelity，但几何上等价）。

4. 新需要的矩阵元：⟨Â φm | Â φn⟩ 和 ⟨Â φm | Â† ψ⟩

展开 Â = 1 + i(dt/2)Ĥ，Â† = 1 − i(dt/2)Ĥ（H 厄米）：

⟨Â φm | Â φn⟩ = ⟨φm|φn⟩ + (dt/2)² ⟨φm|Ĥ²|φn⟩    + i(dt/2) ( ⟨φm|Ĥ|φn⟩ − ⟨Ĥφm|φn⟩ )

最后两项等于 2i(dt/2)·Im⟨φm|Ĥ|φn⟩（H 厄米时）。所以实际要算：

  1. Overlap Smn = ⟨φm|φn⟩ — 已有（overlap()）
  2. H 矩阵元 Hmn = ⟨φm|Ĥ|φn⟩ — 已有（build_HS，各 Hamiltonian term 独立写）
  3. H² 矩阵元 (H²)mn = ⟨φm|Ĥ²|φn⟩ — 要新写
H² 是主要工作量。 对我们 H = T + Vho（+ 后续 Vδ, Vkick）：
H² = T² + TV + VT + V²
其中每个都是 Gaussian 上的多项式期望值，用 Isserlis 定理（Wick's probability theorem）解析算。

4.1 H² 矩阵元的结构

单粒子 1D（N=1）情形，H = p²/2m + (1/2)mω²x²。用 ℏ=m=ω=1：

项算子作用在 Gaussian 上
T−(1/2)∂²x乘出 x² 多项式 × Gaussian（已有实现）
V(1/2)x²直接乘 x²
T²(1/4)∂⁴x乘出 x⁰, x², x⁴ 多项式 × Gaussian
V²(1/4)x⁴直接乘 x⁴
TV+VT−(1/4)(∂²x² + x²∂²)乘出 x⁰, x² 多项式 × Gaussian（应用链式法则）

核心积分（paper 式 20-26 的 Isserlis 定理版本）：对两个 Gaussian φm(x) = e−amx² + 2bmx + cm，φn 类似：

⟨φm|xk|φn⟩ = ⟨φm|φn⟩ × Mk(am+an, bm+bn)

其中 Mk 是关于 k 阶位置矩的封闭公式：⟨x⟩ = b/a, ⟨x²⟩ = 1/(2a) + (b/a)², ⟨x⁴⟩ = 3/(4a²) + 3b²/(a³) + (b/a)⁴ 等。所有 H² 项都是这些基础矩的加权和。

好消息： 我们已有的 calculate_Hamiltonian_kinetic 等函数里已经用过类似的 4 阶矩表达式（T 里有 ∂²ψ → 二阶位置矩 × Gaussian）。新写 T²、V²、TV+VT 是沿着同样的模式推一层。预计 N=1 单粒子约 100-200 行 C++。

4.2 多粒子 (N≥2) 的扩展路径

N 粒子的 H² 需要对所有置换求和（像现有 build_HS 那样循环 PermutationSet）。H² 展开后变成：
(Ti + Vi,j)(Tk + Vk,l) = TiTk + TiVk,l + Vi,jTk + Vi,jVk,l

这是多项式算子作用，Gaussian 上仍封闭。paper 给出 3D/4D Hénon-Heiles 的全套公式（paper §III.A 式 43 附近）。数量级 O(N²) 复杂度，与现有 H 矩阵元匹配。

6.1 BFGS 实现选择

  • A. 写一个简化的 LBFGS（推荐）：50 行 C++，给梯度即可。Rothe 论文 BFGS 一般 20-60 iter 收敛，不需要特别快的实现。
  • B. 调用 CMake 时加 ceres-solver 依赖：工业级，但构建更重。
  • C. 暴力 Gauss-Newton / Levenberg-Marquardt：因为 Rothe 残差是最小二乘形式 r = ||Jα − b||²，LM 往往比 BFGS 快。Eigen 自带 Eigen::LevenbergMarquardt。

初版选 A（自写简化 LBFGS），跑通后再考虑替换。

7. 里程碑

  • M1：H² 矩阵元 & 单元测试（~1 天）
    实现 build_H2（单粒子 H = T + Vho）。测试：用 Gaussian 基态 φ0 的精确本征值 E=0.5 → ⟨φ0|H²|φ0⟩ = E² = 0.25。验证数值精度 1e-10 级。
  • M2：Rothe step for K=1 谐振子基态（~0.5 天）
    起点：K=1 ECG 基态（A=0.5, B=0, R=0, u 归一化）。
    预期：α 优化解就是 α 本身不变（基态是 H 的本征态，Â−1Â† 作用只乘 e−iE·dt 相位），c 变为 e−iE·dt·u。
    验证：1 步 Rothe error ~1e-15 级；T=2π 跑 628 步，|F(T)|² = 1 严格 1.0（K=1 无 gauge 方向，没有任何病态）。
  • M3：Rothe step for K=2-3 踢后态（~1 天）
    拿 task D 的 K=1 基+1 动量副本 → K=2 的踢后态作起点。跑 T=2π 真实时演化。
    预期：E 严格守恒（CN 是幺正的），累积 Rothe error < 1e-3。BFGS 每步 ~20-40 iter。
    验证：|F(T)|² 接近 1（谐振子 recurrence 定理）。
  • M4：K 大一点 + 自适应加/删基（~1 天）
    按 paper 式 50 删 |ci|/max|c| < 1e-6 的基；按残差阈值加新 Gaussian。达到任意 K 的稳定演化。
  • M5：delta + kicking 的 H² 扩展（~2 天）
    多粒子 H = T + Vho + Vδ + Vkick 的 H² 矩阵元。这是做多体 MBDL 量化模拟的最后一块。
