1. 目标

用任务 D 的 Route 1 踢后态作为起点，做实时 TDVP 演化一个谐振周期 T = 2π（H = T̂ + Vho，单粒子 N=1）。

验收：

  • 撑完 T=2π（628 步 @ dt=1e-2，或 6283 步 @ dt=1e-3）
  • E(t) 守恒（H 不含 t，理论严格守恒）
  • Norm 守恒（幺正演化）
  • |⟨ψ(0)|ψ(T)⟩|² ≈ 1（HO recurrence）

2. 理论诊断：TDVP 守恒不守恒？

严格证明：连续 TDVP 下，E 和 ‖ψ‖² 严格守恒。 只依赖两件事：PT 正交投影的厄米性 + |ψ⟩ ∈ TψM（ECG 线性 u 自动满足）。 与「ψ 是否本征态、流形是否完备、基是否线性无关」无关。

两行证明的关键一步（Ĥ PT Ĥ 厄米 ⟹ 期望值为实 ⟹ −i · ℝ 取实部 = 0）：

(Ĥ PT Ĥ)† = Ĥ† PT† Ĥ† = Ĥ PT Ĥ
数值为什么还是飘？因为 C 病态 → 实现用 SVD 伪逆 + Tikhonov (C+λI) 替代真正的 C−1 → 对应的 P̃T 不再厄米、也不再是投影 → 上面证明里「Ĥ PT Ĥ 厄米」那一步断了 → E 出现虚部速率项 → 飘。

K=1 纯基态自检（task6）跑 628 步全部机器零误差，说明核心 TDVP 装配（矩阵元、∂αψ、P⊥、RK4）没有 bug。所有 K≥3 的失败都归结到「过参数化 + C 近奇异 + 伪逆破对称」。

5. 当前成功点：Wiener + 小 dt

Wiener 平滑 sinv = sv / (sv² + λ²) 的三个性质：

  • sv=0 严格给 0 ⟹ 真 gauge 方向精确清零（必须 λ_C=0）
  • sv 连续函数 ⟹ 无阶梯跳
  • 有限放大 1/(2λ) ⟹ 比硬 rcond 温和

代价是有偏差（不是 TDVP 严格解），每步 O(λ²/sv)，积累与 dt 成正比。所以 dt 必须小：

  • dt=1e-2：bias 直接把态推到错位 manifold，E 锁在 5.96 失败
  • dt=1e-3：bias 可控，E 恒定 4000+ 步到 6 位小数 成功
