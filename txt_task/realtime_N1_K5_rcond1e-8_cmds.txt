Task: Real-time TDVP, N=1 K=5, full HO period (T = 2*pi), dt = 1e-3, rcond = 1e-8

Purpose
- Test whether the K=5 dynamics failure (50% norm leak, parity violation
  to 0.18, peak-height disagreement up to 50% — see useful_result_April/
  N1_realtime_K3_vs_K5_explanation.txt) is caused by SVD truncation at
  the default rcond=1e-4.
- The N=1 K=5 step had cond(C) ~ 1e8, so default rcond=1e-4 dropped about
  4 decades of singular directions, including parity-symmetric tangent
  flow. rcond=1e-8 keeps essentially everything.
- If rcond=1e-8 fixes the dynamics (peak heights match grid, |dz| stays
  small, norm flat), we have proof the failure was over-aggressive
  truncation, not basis size.
- NOT enabled: --enforce-norm, --u-split-trotter (per user request).

CRITICAL: backup existing K=5 trace files before running -- the CSV
names depend on N and K only, not on rcond, so this run will overwrite
the default-rcond results. Run:

  mkdir -p out/verify/rcond_default
  cp out/verify/step4_ecg_trace_N1_K5.csv \
     out/verify/step4_ecg_density_N1_K5.csv \
     out/verify/step4_ecg_nk_N1_K5.csv \
     out/verify/step4_ecg_snap_N1_K5.csv \
     out/verify/rcond_default/

Pre-flight: pick a free core (filtered to your allowed set)

ALLOWED=$(taskset -pc $$ | awk -F'list: ' '{print $2}')
echo "your allowed cores: $ALLOWED"

top -bn2 -d 1 -1 2>/dev/null | awk -v allowed="$ALLOWED" '
  BEGIN {
    n=split(allowed, parts, ",")
    for (i=1;i<=n;i++) {
      if (split(parts[i], r, "-") == 2) for (c=r[1]+0;c<=r[2]+0;c++) ok[c]=1
      else ok[parts[i]+0]=1
    }
    snap=0
  }
  /^top - / { snap++ }
  snap==2 && /^%Cpu[0-9]/ {
    c=$1; gsub(/^%Cpu/,"",c); gsub(/[^0-9]/,"",c)
    if ((c+0) in ok) {
      for (i=1;i<=NF;i++) if ($i=="id,") { idle=$(i-1); break }
      printf "cpu %3d  used=%5.1f%%  idle=%5.1f%%\n", c, 100-idle, idle
    }
  }
' | sort -k3 -t= -nr

How to run

cd /home/gyqyan/zexuan/ecg1d_cpp

mkdir -p logs && env OMP_NUM_THREADS=1 OPENBLAS_NUM_THREADS=1 MKL_NUM_THREADS=1 \
nohup taskset -c 240 ./build/ecg1d_verify \
  --N 1 --K 5 --run-step4 \
  --T 6.283185307 --dt 1e-3 --dt-trace 1e-3 \
  --n-snap 11 --moment-form both --rt-verbose \
  --rcond 1e-8 \
  > logs/realtime_N1_K5_rcond1e-8.log 2>&1 &
echo "PID=$!"
