## 3. 关键技术发现

### 3.1 Ghost State Bug（已修复）

**Eigen aliasing bug**：`A = 0.5*(A + A.transpose())` 在 Eigen 中不会原地对称化（惰性求值）。必须加 `.eval()`。

此 bug 导致 stochastic refine 中 H 矩阵不满足 Hermitian 对称性 → 变分原理被打破 → 出现低于基态的虚假能量。修复后 refine 完全正常。

### 3.2 B=0, R=0 对静态优化的重要性

基态是实函数、零动量、中心在原点。SVM 阶段搜索非零 B 和 R 是纯浪费：

| | 含 B/R | 仅 A |
|---|---|---|
| 参数维度 (N=2) | 7 | 1 |
| 有效搜索空间 | 7 维中找 1 维子空间 | 1 维直接搜索 |
| SVM 效率 | 低 | 高 |

### 3.3 C 矩阵条件数与 TDVP 效率

```
cond(C) 与有效自由度:

         K=33 kicking:  cond(C) = 3.4e9,  rank = 13/99
         K=20 gaussian:  cond(C) = 3.1e8,  rank = 11/60
         K=12 delta:     cond(C) = 2.9e8,  rank = 7/36

99 个 A 参数中 86 个对 TDVP 是冗余的（改变它们对波函数几乎没影响）。
```

## 4. 实时间 Kicked Dynamics 进展

### 4.1 目标

模拟 N=2, gamma=0（无相互作用）的量子 kicked rotor，与有限差分精确解对比。这是验证实时间演化管线的第一步，为最终模拟 many-body dynamical localization（Science 2025 论文）做准备。

### 4.2 已实现的组件

| 组件 | 状态 | 说明 |
|------|------|------|
| 解析 kick 算符 | 完成 | Jacobi-Anger/Bessel 展开，精确投影 |
| 有限差分精确解 | 完成 | 单粒子 kicked rotor，N=2 = 2×单粒子 |
| 固定基底传播 | 完成 | u(t) = exp(-iS^{-1}Ht)u(0)，精确 |
| 动量增广基底 | 完成 | 贪心 S 条件数筛选 |
| 实时间 TDVP | **不稳定** | 文献确认这是公认难题 |

### 4.3 当前最佳结果

方法：动量增广基底（K=17）+ 固定基底精确传播

```
能量 E(n)
  |
6 ─                                         ECG (K=17)
  |        *                              *---*---饱和
5 ─       / \                           /
  |      /   \                         /
4 ─     /     *                       *
  |    /       \                     /
3 ─   /         *                   *
  |  /           \                 /
2 ─ *              *             *
  |                  \         /
1 ─ *---*---*---*------*-----*                  ← Exact (振荡 1.0~1.7)
  |
  └──────────────────────────────────────
    kick 0   5   10   15   20   25   30   50
```

| Kick | ECG E | 精确 E | 相对误差 |
|------|-------|--------|---------|
| 0 | 1.000 | 1.000 | 0% |
| 1 | 1.388 | 1.315 | 5.5% |
| 3 | 1.649 | 1.636 | **0.8%** |
| 5 | 1.555 | 1.073 | 45% |
| 10 | 2.524 | 1.094 | 131% |
| 50 | 4.126 | 1.737 | 138% |

前 3 个 kick 误差 <10%，但之后能量漂移。原因：每次 kick 投影丢失高动量成分，误差逐 kick 累积。

### 4.4 下一步计划

1. **分步法**：u 精确演化 + z 虚时间优化（分离良态和病态部分）
2. **Rothe 方法**：每步重新优化所有参数（文献 SOTA，Schrader et al. 2025）
3. 验证通过后：加入相互作用（gamma > 0），观察 many-body dynamical localization 信号
